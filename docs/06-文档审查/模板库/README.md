## 技术文档模板

### API文档模板

```markdown
# API接口文档

## 概述

[API的基本信息和用途]

## 接口列表

### 接口名称

**接口地址**: `[API地址]`  
**请求方法**: `GET/POST/PUT/DELETE`  
**请求格式**: `JSON/XML/Form-Data`  
**响应格式**: `JSON/XML`  
**认证方式**: `Bearer Token/Basic Auth`  

#### 请求参数

| 参数名 | 类型 | 必填 | 默认值 | 说明 |
|-------|------|------|--------|------|
| param1 | string | 是 | - | 参数1说明 |
| param2 | number | 否 | 100 | 参数2说明 |

#### 响应示例

```json
{
  "code": 200,
  "message": "success",
  "data": {
    "key": "value"
  }
}
```

#### 错误码说明

| 错误码 | 说明 | 解决方案 |
|-------|------|----------|
| 400 | 请求参数错误 | 检查请求参数格式和内容 |
| 401 | 认证失败 | 检查认证信息是否正确 |
| 403 | 权限不足 | 联系管理员获取权限 |
| 500 | 服务器内部错误 | 联系技术支持 |
