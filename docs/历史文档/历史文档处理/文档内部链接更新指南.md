   ```
   ../01-架构设计/
   ../02-开发指南/
   ../03-测试文档/
   ../05-FixBug/
   ```

2. **替换模式**:
   ```
   ../01-项目概述/
   ../02-架构设计/
   ../03-开发指南/
   ../04-测试文档/
   ../05-Bug修复/
   ```

### 方法2: 正则表达式替换
使用正则表达式进行精确替换：

```regex
# 匹配相对路径链接
\.\.\/00-项目概述\/([^)]+)
\.\.\/01-架构设计\/([^)]+)
\.\.\/02-开发指南\/([^)]+)
\.\.\/03-测试文档\/([^)]+)
\.\.\/05-FixBug\/([^)]+)

# 替换为
./01-项目概述/$1
./02-架构设计/$1
./03-开发指南/$1
./04-测试文档/$1
./05-Bug修复/$1
```
