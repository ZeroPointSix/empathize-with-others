# 代码审查快速检查清单

## 通用检查（所有项目）

### 代码质量
- [ ] 无魔法数字
- [ ] 无硬编码配置
- [ ] 函数职责单一
- [ ] 无重复代码
- [ ] 命名清晰

### 错误处理
- [ ] 所有异常都被处理
- [ ] 错误信息清晰
- [ ] 不吞掉错误
- [ ] 关键错误有日志

### 安全性
- [ ] 用户输入都验证
- [ ] 无注入风险
- [ ] 无敏感信息泄露
- [ ] 有权限检查

---

## 后端项目附加检查

### 架构
- [ ] 路由→控制器→服务→仓库分层
- [ ] 路由中无业务逻辑
- [ ] 服务层无HTTP依赖
- [ ] 仓库层无业务逻辑

### 数据库
- [ ] 使用参数化查询
- [ ] 有事务处理（如需要）
- [ ] 无N+1查询
- [ ] 连接正确关闭

### API
- [ ] 有输入验证
- [ ] 返回合适的HTTP状态码
- [ ] 错误响应格式统一
- [ ] 有API文档（如有必要）
