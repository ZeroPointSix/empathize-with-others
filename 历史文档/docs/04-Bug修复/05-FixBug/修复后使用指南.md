# 修复后使用指南

**版本**：1.0.0  
**更新日期**：2025-12-05

---

## 🎯 快速开始

本指南帮助你快速上手使用修复后的应用核心功能。

---

## 1️⃣ 首次使用：配置 API Key

在使用 AI 功能之前，你需要先配置 API Key。

### 步骤：

1. **打开应用**，进入联系人列表页面
2. **点击右上角的设置图标** ⚙️
3. **输入你的 API Key**
   - 支持 OpenAI、DeepSeek、Google Gemini
   - API Key 将使用硬件加密安全存储
4. **点击"保存 API Key"按钮**
5. **选择 AI 服务商**（可选）
   - 点击"当前服务商"卡片
   - 选择你想使用的服务商

### 提示：
- ✅ API Key 加密存储，不会上传到任何服务器
- ✅ 可以随时删除或更换 API Key
- ✅ 支持显示/隐藏 API Key

---

## 2️⃣ 添加联系人

### 步骤：

1. **在联系人列表页面**，点击右下角的 **"+" 按钮**
2. **填写联系人信息**：
   - **姓名**（必填）：联系人的名字
   - **电话**（可选）：联系人的电话号码
   - **目标关系**（可选）：你与此人的沟通目标，例如"建立良好的合作关系"
3. **点击"添加"按钮**保存
4. 新联系人将立即显示在列表中

### 提示：
- ✅ 姓名是必填项
- ✅ 电话和目标关系可以稍后编辑
- ✅ 点击"取消"不会保存数据

---

## 3️⃣ 查看和编辑联系人

### 查看联系人详情：

1. **在联系人列表中**，点击任意联系人
2. 查看联系人的详细信息：
   - 基本信息（姓名、目标关系）
   - 上下文深度
   - 脑标签（如果有）

### 编辑联系人：

1. **在联系人详情页**，点击右上角的 **编辑图标** ✏️
2. **修改信息**
3. **点击"保存"按钮**或右上角的保存图标

### 删除联系人：

1. **在联系人详情页**，点击右上角的 **删除图标** 🗑️
2. **确认删除**

---

## 4️⃣ 使用 AI 分析对话

这是应用的核心功能！

### 步骤：

1. **进入联系人详情页**
2. **点击"分析对话"按钮**
3. **输入聊天内容**：
   - 可以输入多条消息
   - 区分"我"和"对方"的消息
4. **点击右上角的 AI 图标** 🧠 开始分析
5. **查看 AI 分析结果**：
   - 情感分析
   - 沟通建议
   - 回复建议
   - 风险提示

### 提示：
- ⚠️ 需要先配置 API Key 才能使用
- ✅ 敏感信息会自动掩码保护隐私
- ✅ 可以直接应用 AI 的回复建议

---

## 5️⃣ 隐私设置

### 数据掩码

**作用**：AI 分析前自动掩码敏感信息（姓名、电话、地址等）

**如何开启**：
1. 进入设置页面
2. 找到"隐私设置"区域
3. 打开"数据掩码"开关

### 本地优先模式

**作用**：优先使用本地规则，减少 AI 调用，节省费用

**如何开启**：
1. 进入设置页面
2. 找到"隐私设置"区域
3. 打开"本地优先模式"开关

## 🔧 常见问题

### Q1: 为什么点击"分析对话"没有反应？

**A**: 请检查是否已配置 API Key：
1. 进入设置页面
2. 查看是否显示"API Key 已配置"
3. 如果没有，请先配置 API Key

### Q2: API Key 安全吗？

**A**: 非常安全！
- ✅ 使用 Android 硬件加密存储
- ✅ AES-256-GCM 加密算法
- ✅ 不会上传到任何服务器
- ✅ 只在本地使用

### Q3: 如何更换 API Key？

**A**: 
1. 进入设置页面
2. 点击"API Key 已配置"卡片右侧的删除图标
3. 输入新的 API Key
4. 点击"保存 API Key"

### Q4: 支持哪些 AI 服务商？

**A**: 目前支持：
- OpenAI（GPT-4、GPT-3.5）
- DeepSeek
- Google Gemini

### Q5: 如何切换 AI 服务商？

**A**:
1. 进入设置页面
2. 点击"当前服务商"卡片
3. 选择你想使用的服务商
4. 系统会自动保存

## 🎓 使用技巧

### 技巧 1：充分利用目标关系

在添加联系人时，填写清晰的"目标关系"可以帮助 AI 提供更精准的建议。

**示例**：
- ❌ 不好："朋友"
- ✅ 更好："希望建立长期合作关系的商业伙伴"

### 技巧 2：提供完整的对话上下文

在使用 AI 分析时，提供更多的对话历史可以获得更准确的分析。

### 技巧 3：定期更新联系人信息

随着你对联系人了解的加深，及时更新他们的信息和标签，AI 的建议会越来越准确。

### 技巧 4：善用数据掩码

如果担心隐私问题，建议开启"数据掩码"功能，AI 分析时会自动隐藏敏感信息。
