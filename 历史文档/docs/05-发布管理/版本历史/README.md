# 版本历史

本目录记录项目的版本发布历史和变更日志。

## 版本记录格式

每个版本发布应包含以下信息：
- 版本号
- 发布日期
- 主要变更
- Bug修复
- 新增功能
- 改进进项
- 已知问题

## 版本命名规范

遵循主版本号.次版本号.修订号-环境标识的格式：
- 主版本号：重大功能变更或架构调整
- 次版本号：新功能添加或重要改进
- 修订号：Bug修复和小改进
- 环境标识：dev(开发)、test(测试)、release(发布)
