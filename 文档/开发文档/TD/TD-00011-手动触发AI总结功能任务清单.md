## 12. 开发注意事项

### 12.1 核心原则

1. **复用优先**：最大化复用现有 AiSummaryProcessor 和 LocalSummaryProcessor
2. **状态不可变**：使用 data class 和 StateFlow 管理状态
3. **可取消性**：所有长时间操作支持协程取消
4. **向后兼容**：扩展现有模型，不破坏已有功能

### 12.2 关键依赖

- 依赖 TDD-00003 的 DailySummary 模型和 ConversationLog 模型
- 依赖 TDD-00007 的 ConversationContextBuilder
- 复用现有的 AiSummaryProcessor 和 LocalSummaryProcessor

### 12.3 V1 简化说明

- SummaryTaskManager 在 V1 版本中不单独实现，任务状态管理直接在 ManualSummaryViewModel 中处理
- V1.1 版本再考虑抽取独立的任务管理器以支持任务队列、并发控制等高级功能

### 12.4 性能要求

| 指标 | 目标值 |
|------|--------|
| 日期选择器响应 | < 100ms |
| 冲突检测 | < 500ms |
| 30天范围总结 | < 30s |
| UI帧率 | ≥ 55fps |
| 内存增量 | < 10MB |
