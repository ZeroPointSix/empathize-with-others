## 🎨 设计风格说明

所有原型严格遵循 `HTML/联系人/` 目录下的设计规范：

### 色彩规范
- 统一背景色：`#F2F2F7`（iOS系统灰）
- 卡片背景：`bg-white`
- 主色调：
  - 概览页：`slate-900`（深色文字）+ `blue-500`（强调色）
  - 事实流：`#007AFF`（iOS蓝）
  - 画像库：`#07C160`（微信绿）
  - 资料库：`indigo-600`（靛蓝）

### 卡片样式
- 圆角：`rounded-[20px]` 或 `rounded-[24px]`
- 阴影：`shadow-[0_8px_30px_-12px_rgba(0,0,0,0.06)]`
- 边框：`border border-slate-100`

### 顶部导航
- 统一使用圆形按钮：`rounded-full bg-white/50 backdrop-blur-md`
- 毛玻璃效果：`bg-[#F2F2F7]/85 backdrop-blur-xl`

### Tab 指示器
- 概览页：底部黑色下划线 `w-8 h-[3px] bg-slate-900`
- 事实流：底部蓝色下划线 `w-8 h-[3px] bg-[#007AFF]`
- 画像库：底部绿色下划线 `w-8 h-[3px] bg-[#07C160]`
- 资料库：底部靛蓝下划线 `w-8 h-[3px] bg-indigo-600`

### 底部导航
- 毛玻璃效果：`bg-white/80 backdrop-blur-xl`
- 悬浮按钮：圆形，品牌色背景，带阴影

### 动画效果
- 淡入动画：`@keyframes fadeIn` + `animation-delay` 实现错落效果
- 交互反馈：`active:scale-[0.98]` 或 `active:scale-95`
- 过渡效果：`transition-all duration-300`

## 📋 页面功能说明

### 概览页 (OverviewTab)
- 大头像展示 + 在线状态指示
- 关系健康度环形图表
- 核心标签引用样式展示
- 最新发现卡片
- 专属AI指令入口
- 沟通目标展示

### 事实流页 (FactStreamTab)
- 时光轴/清单视图切换
- 情绪类型筛选（甜蜜/冲突/日期/礼物/里程碑）
- 对话记录卡片（含AI建议）
- AI日总结卡片
- 里程碑卡片
- 用户事实卡片
- 礼物记录卡片
- 手动触发AI总结悬浮按钮

### 画像库页 (PersonaTab)
- 搜索栏
- 个性化标签区域（可添加/删除）
- 雷区标签区域（高/中/低风险）
- 策略标签区域（推荐操作）
- AI自动推测区域（采纳/拒绝）
- 添加标签悬浮按钮

### 资料库页 (DataVaultTab)
- 搜索栏
- 数据统计摘要卡片
- 数据来源网格（聊天/AI总结/媒体/语音/视频/文档）
- 最近活动列表
- 导入数据悬浮按钮

### 新建联系人页 (CreateContactScreen)
- iOS标准导航栏（取消/完成）
- 头像上传区域
- 姓名输入
- 沟通目标输入
- 添加画像事实按钮
- 删除联系人按钮
