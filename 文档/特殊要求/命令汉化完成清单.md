# 斜杠命令汉化完成清单

## 汉化日期
2024-12-12

## 汉化范围
对 `.kiro/commands/`、`.claude/commands/` 和 `.Roo/commands/` 三个目录下的所有斜杠命令文件进行汉化。

## 汉化原则
1. 在 YAML front matter 的 `description` 字段添加中文描述
2. 在文件开头添加 `**中文说明：**` 段落，提供简短的中文功能说明
3. 保持命令的英文执行逻辑不变
4. handoffs 标签也进行汉化

## 已汉化的命令文件

- **位置**: `.kiro/`, `.claude/`, `.Roo/`

#### Gitcommit.md

## 注意事项

1. **命令执行逻辑未改变**: 所有英文的执行步骤、规则和约束保持不变
2. **仅用户可见部分汉化**: 只汉化了 description 和说明文字，不影响命令功能
3. **三个目录同步**: `.kiro/`、`.claude/`、`.Roo/` 三个目录的命令保持一致
4. **Handoffs 汉化**: 命令间的跳转提示也已汉化，提升用户体验
